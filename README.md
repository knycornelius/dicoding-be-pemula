# Dicoding Bootcamp

Final project submission for Belajar Membuat Apliakasi Back-End untuk Pemula Bootcamp

## Technologies

- Javascript
- NodeJS
- Hapi Framework
- Postman

## How to start

- Run server
  ```
  yarn start
  ```
- Format codebase
  ```
  yarn format
  ```

## How to test

- Import the Postman Collection and Postman Environment in Postman app
- Run the tests in sequence
